"Optimizing Images for the Web"


Optimizing images can be a very important thing to do in order to create an optimal user experience for users that want to be able to veiw the website without having to wait for text and images to load in the browser.

There are other benefits to optimizing the images on websites. Some of them include better SEO (search engine optimization) and faster loading speeds.

The reason that websites that have optimized images load faster than website that do not have optimized images is becasue the optimizer tool removes data that is not necessary for the user to veiw it on the webpage. This make the loading process a lot smoother and faster.


The image on the home page of this website had an original size of width: 852px, height:1136px and image file size of 122.9kb. In order to save bytes and make the website load faster I optimized this image with an image optimization tool on the jpeg-optimizerjpeg-optimizer  website.

The results after the image was optimized was width:300px, height:400px and a file size of 14kb. The total savings in the file size was 88.6% and that is a lot when you think about it.
